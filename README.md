MongoDB
=======

- Download the 64-bit version from: 

http://mongodb.org/downloads

- Run the Mongo DB server: 

/mongodb-osx-x86_64-2.4.9/bin/mongod

- Note: The database will be stored in: /data/db by default

- Mongo DB commands: 

> - db
> - show dbs
> - use [DatabaseName]
> - show collections
